# commentgen

### Command Help

```shell
Usage: commentgen [OPTIONS] TEXT

Options:
  --lang TEXT
  --length INTEGER
  --pattern TEXT
  --help            Show this message and exit.
```

### Example:

```java
// $ commentgen --pattern "~" --lang java "start of  hello-world"

/* ~~~~~~~~~~~~~~~~~~~~~~~~~~ start of  hello-world ~~~~~~~~~~~~~~~~~~~~~~~~~ */
```
